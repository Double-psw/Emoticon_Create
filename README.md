# Emoticon_Create
------
## Stable diffusion을 이용한 감성 이모티콘 개인화 서비스 

- app => Flask+React를 활용한 웹어플리케이션 개발이 담겨있는 폴더 
- model => Stable Diffusion Fine tunning에 대한 과정이 담겨 있는 폴더 
- data => 사용한 데이터에 대한 설명과 데이터가 담겨 있는 폴더
